Using the tools library
===================

Geobatch is developed leveraging on a huge set of tools.
Tools is a modular subproject of Geobatch which can be located here:

	${GEOBATCH}/src/tools/

Here is the list of available modules splitted by dependencies:

Tools modules
^^^^^^^^^^^^^

.. toctree::
   :maxdepth: 1
   :glob:

   tools*

Other (Undocumented) tools
^^^^^^^^^^^^^^^^^^^^^^^^^^

  * commons - Commons utilities without specific dependencies
  * io - I/O based utilities
  * ant -  Apache Ant based utilities
  * compress - compress/extract tools
  * freemarker
  * geotools
  * netcdf
  * geobatch - geobatch specific tools
